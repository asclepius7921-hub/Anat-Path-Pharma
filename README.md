# Anat Path Pharma 🩺

A flashcard PWA for NEET-PG and FMGE revision.

## 755 cards across 3 subjects

- 🔬 Pathology — 208 cards, 94 PYQ tagged

## Features
- Tap to flip · Swipe to sort
- ⭐ PYQ-only filter for exam drilling
- Progress tracking across sessions
- Works fully offline after first visit
- Add to Home Screen on any phone

## Use it
Open the link on your phone and tap **Add to Home Screen**.
